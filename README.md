# Isomorphic Vite App

This app has been converted to a local-only Vite React project.

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run start
```

Server-side Next.js, NextAuth, UploadThing, SMTP, pnpm, and Turbo wiring have been removed from this app. Upload and email actions use local dummy behavior.
